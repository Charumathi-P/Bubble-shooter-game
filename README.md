# 🎯 Bubble Shooter Game
A classic Bubble Shooter game developed using **HTML**, **CSS**, and **JavaScript**. Match bubbles of the same color, clear the board, and achieve the highest score!

---

## 🚀 Features
- 🎮 Classic Bubble Shooter gameplay
- 🎯 Mouse aiming and shooting
- 🌈 Multiple bubble colors
- 💥 Bubble collision detection
- ⭐ Score tracking
- 🔄 Restart game
- 📱 Responsive design
- ⚡ Smooth animations

---

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- HTML5 Canvas
  
---

## ▶️ How to Run
1. Clone the repository
2. Open the project folder.
3. Run `index.html` in your browser.
Or open the project using **VS Code** and use the **Live Server** extension.

---

## 🎮 How to Play
- Aim using your mouse.
- Click to shoot a bubble.
- Match three or more bubbles of the same color.
- Clear all bubbles to win.
- Try to achieve the highest score!
